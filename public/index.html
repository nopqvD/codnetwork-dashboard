💡 What are Webhooks? ▶ Click to expand
Webhooks are automatic notifications sent by CODNETWORK to your dashboard whenever an order event happens (e.g., a status change, a delivery, or a return). Instead of constantly polling for updates, webhooks push real-time data to your server.

How it works: When an event occurs on CODNETWORK, it sends an HTTP POST request to your webhook URL. The dashboard processes the event, updates inventory deltas, and logs the event.

Inventory Deltas: When an order is confirmed/shipped/delivered, the webhook subtracts the product quantity from inventory. When an order is returned, it adds the quantity back.

Where to add it: In CODNETWORK, go to Settings → Webhooks → Add URL and paste your webhook URL (shown below).

Webhook URL format:
https://your-domain.com/webhook?token=YOUR_API_TOKEN

What data is captured: Order status changes (confirmed, shipped, delivered, returned) → inventory delta is computed and saved. All events are persisted to db.json on the server, so data survives restarts.

Copy this URL and paste it in your CODNETWORK settings under Webhook URL.
