What are Webhooks?
Webhooks are automatic notifications sent by CODNETWORK to your dashboard whenever an order event happens (e.g., a new order, a status change, a delivery, or a return). Instead of constantly polling for updates, webhooks push real-time data to your server.

How it works: When an event occurs on CODNETWORK, it sends an HTTP POST request to your webhook URL (/webhook?token=YOUR_TOKEN). The dashboard processes the event, updates inventory deltas, and logs the event below.

Inventory Deltas: When an order is confirmed/shipped/delivered, the webhook subtracts the product quantity from inventory. When an order is returned, it adds the quantity back.
